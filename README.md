# weflow 项目模版备份

## 项目结构

```
└── src
  ├── html                页面
      ├── template.html  html
  ├── img               图片
  ├── js               js文件
    ├── api              接口文件
    ├── rem              rem适配js
    ├── util             js工具库
    ├── plugin           插件
  ├── css              样式库
    ├── base                  基本样式模块
    ├── style-main.scss       公共样式 (需要添加style前缀才能编译)
    ├── style-template.scss   模块样式

```
